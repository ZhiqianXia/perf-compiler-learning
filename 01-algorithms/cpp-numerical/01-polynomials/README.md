# 第1章 多项式的计算

## 背景
V5 的第 1 章集中在多项式求值、求导、相乘、相除与复系数扩展。对应旧文件包括 `1BPLY.C`、`1CPLY.C`、`1PDIV.C`、`1PLYS.C`。

## 代表算法
本章先保留最核心、最稳健的 Horner 法。它把
`p(x) = a_0 + a_1 x + ... + a_n x^n`
重写为嵌套形式，从而把求值复杂度从显式幂计算的 $O(n^2)$ 风格降到严格的 $O(n)$。

## 为什么先实现它
- 是多项式数值计算的基础模块
- 可直接扩展到导数同时求值
- 稳定性优于先算幂再线性组合

## 旧代码到现代 C++ 的整理
- 旧版特点：裸数组、过程式接口、默认列主序/隐式约定多
- 新版改写：`std::vector<double>`、返回值语义、异常处理、示例驱动

## 文件
- `polynomial_horner.cpp`: Horner 求值与导数同时求值
- `polynomial_division.cpp`: 对应 `1PDIV.C` 的多项式长除法
- `polynomial_multiply.cpp`: 对应 `1PLYS.C` 的多项式相乘
