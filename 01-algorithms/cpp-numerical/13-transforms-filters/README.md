# 第13章 数学变换与滤波

## 背景
V5 第 13 章包括傅里叶级数、快速傅里叶变换、沃尔什变换和数字滤波。代表旧文件包括 `13FOUR.C`、`13KFFT.C`、`13KFWT.C`、`13LMAN.C`。

## 代表算法
这里先保留离散傅里叶级数系数的计算，它和连续傅里叶分析相比更适合当前仓库的教学定位，也和旧版 `13FOUR.C` 直接对应。

## 背景知识
设在等距网格上采样得到 $f_j$，则傅里叶级数系数可以看作基函数
$1, \cos(kx), \sin(kx)$
上的离散投影。

## 文件
- `fourier_series.cpp`: 离散傅里叶级数前若干系数计算
- `fft.cpp`: Cooley-Tukey 迭代 FFT，对应 `13KFFT.C`
