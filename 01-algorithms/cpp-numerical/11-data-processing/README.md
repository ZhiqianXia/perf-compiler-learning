# 第11章 数据处理

## 背景
V5 第 11 章包含线性回归、多元回归、逐步回归、相关分析。代表旧文件有 `11SQT1.C`、`11SQT2.C`、`11LOG1.C`。

## 代表算法
这里以一元线性回归为入口。旧版 `11SQT1.C` 已经给出了截距、斜率和残差统计量的计算逻辑，现代版则整理为结果结构体。

## 背景知识
最小二乘的目标是最小化
$$
\sum_{i=1}^{n} (y_i - (\beta_0 + \beta_1 x_i))^2
$$
闭式解来自正规方程，也可看成把数据投影到二维仿射子空间。

## 文件
- `linear_regression.cpp`: 一元线性回归与残差指标
- `multiple_linear_regression.cpp`: 多元线性回归的正规方程实现
- `correlation_analysis.cpp`: 协方差与相关系数分析
